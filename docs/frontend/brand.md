# Brand

Visual identity for Clean Up - Give Back.

## Colors

| Token | Value | Usage |
|-------|-------|-------|
| Forest Green (primary) | `#009540` | CTAs, brand accents |
| Light header | `#E8F5E9` | Light mode headers |
| Dark header | `#1B4332` | Dark mode headers |

## Typography

| Role | Font |
|------|------|
| Display / headings | Sanchez |
| Body | Noto Sans |
| Monospace (legacy) | SpaceMono |

## Components

Use `ThemedView`, `ThemedText`, and shared theme hooks (`useColorScheme`, `useThemeColor`). See [context/components.md](context/components.md).

## Copy tone

- Clear, encouraging, community-focused
