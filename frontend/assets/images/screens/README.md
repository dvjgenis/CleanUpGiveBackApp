# Screen rasters

Per-flow illustrations and backgrounds. Mirror the screen keys used under `assets/figma/`.

| Folder | Used by |
|--------|---------|
| `session-setup/` | Guide + steps 2–5 illustrations; complete celebration star/smiley |
| `permissions/` | Location pin + camera graphics (setup steps 6–7) |
| `photo-checkpoint/` | Checkpoint prompt background |
| `photo-submitted/` | Submitted success background |
| `missed-checkpoint/` | Restart-required background |
| `submission-confirmation/` | Mock checkpoint photo thumbs |
| `session-detail/` | Map hero + photo evidence stubs |

SVG icons for the same screens live under `assets/figma/<screen>/`.
